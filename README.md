# Personal Portfolio

A modern, responsive portfolio website built with React and Vite.

## Features

- About Me section
- Portfolio showcase
- Contact form
- Resume/Skills section
- Responsive design
- Modern UI/UX

## Technologies Used

- React
- React Router
- Material-UI
- Vite
- JavaScript

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

To build for production:
```bash
npm run build
```
